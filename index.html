APIs Públicas para Estudo

Aqui você pode encontrar algumas APIs públicas gratuitas para estudar e integrar aos seus projetos.

1. JokeAPI (Piadas de Programação)

Fornece piadas de programação para usar em projetos ou apenas para descontrair.

Visite a JokeAPI

Exemplo de uso:

        fetch('https://v2.jokeapi.dev/joke/Programming?type=single')
            .then(response => response.json())
            .then(data => {
                console.log(data.joke);
            });

2. OpenWeatherMap (Clima)

Obtenha informações sobre o clima em qualquer cidade ao redor do mundo.

Visite o OpenWeatherMap

Exemplo de uso:

        const apiKey = 'SUA_CHAVE_AQUI';
        const city = 'São Paulo';
        const url = \`https://api.openweathermap.org/data/2.5/weather?q=\${city}&appid=\${apiKey}&units=metric&lang=pt_br\`;
        
        fetch(url)
            .then(response => response.json())
            .then(data => {
                console.log(\`Tempo em \${city}: \${data.weather[0].description}\`);
            });

3. Stack Exchange API (Perguntas e Respostas)

Permite buscar perguntas e respostas do Stack Overflow e outros sites da rede Stack Exchange.

Visite a Stack Exchange API

Exemplo de uso:

        fetch('https://api.stackexchange.com/2.3/questions?order=desc&sort=activity&tagged=javascript&site=stackoverflow')
            .then(response => response.json())
            .then(data => {
                console.log(data.items[0].title); // Exibe o título da primeira pergunta
            });
